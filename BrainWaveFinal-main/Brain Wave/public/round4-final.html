🧠 Round 4: Take-Home Project

Problem Statement:

Build a small interactive website for the GMU Coding Club that showcases the club's activities, events, projects, and resources. Your website should be visually appealing, easy to navigate, and demonstrate creativity.

Requirements:

  • A homepage with club information, mission, and navigation links
  • Section for upcoming events or competitions
  • Responsive design: works on desktop and mobile
  • Contact or registration form for new members
  • At least one technical interactive feature (quiz, live code snippet, or animation)
  • Integration with social media links (Facebook, LinkedIn, Instagram)
  • Newsletter or announcements subscription

Projects:

  • Section showcasing previous coding club projects and student contributions
  • Interactive demos or code snippets for each project
  • Ability to filter projects by category or technology

Be creative, use modern web technologies, and make the website engaging for students and visitors.

Submission Deadline: Wednesday, 15 October, 5:00 PM. Teams failing to submit within the given time will not be considered.

⚠️ Note: Do not submit anything here. You will present your website at the final session.

Good luck! Plan well, implement the features carefully, and prepare a clear demo for the judges.
