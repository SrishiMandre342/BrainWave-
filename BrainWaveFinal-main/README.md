# 🧠 BrainWave – Intercollege Coding Competition Platform

BrainWave is a web-based platform developed to conduct and manage an intercollege coding competition with multiple rounds. This project was built and used during a live event where participants progressed through various technical challenges.

---

## 🚀 About the Project

BrainWave was created to streamline the execution of a coding competition consisting of multiple rounds such as quiz, debugging, code optimization, and final project development.

As the **Head Coordinator**, I was responsible for both organizing the event and developing this platform to ensure smooth flow and management of participants.

---

## 👨‍💻 My Role

- Head Coordinator of BrainWave intercollege competition  
- Designed and developed the complete website  
- Managed event execution and technical workflow  
- Handled participant experience and round transitions  

---

## 🛠️ Tech Stack

- HTML

---

## 📂 Project Structure

- `index.html` → Landing page  
- `admin-login.html` → Admin authentication  
- `dashboard.html` → Admin dashboard  
- `leaderboard.html` → Displays participant rankings  

### 🧩 Competition Rounds

- `round1-quiz.html` → Quiz round  
- `round2-debug.html` → Code debugging round  
- `round3-optimize.html` → Code optimization round  
- `round4-final.html` → Final problem statement round  

### ⏳ Utility Pages

- `waiting.html`, `waiting2.html`, `waiting3.html`, `waiting4.html` → Waiting screens between rounds  
- `test-round1.html` → Testing page
